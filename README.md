## Introduction

![Cover](Cover.png)

A [*Kraken Futures*](https://futures.kraken.com) API demo.

## Getting Started

### Prerequisites

- Install [*Go*](https://golang.org).
- Set the API key in the `authkey` folder.

### Building

```bash
go build -tags debug
```

Or

```bash
go build -tags release
```
